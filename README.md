# wmenu
A dmenu clone for windows operating system written with pure win32 API.

# Prerequisites
- MSVC

# Tips
Sicne wmenu uses WINDOWS subsystem, you can't use stdout directly.
So you should call wmenu like this: `.\wmenu.exe | more`
